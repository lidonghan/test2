Chapter 1

起步

本章介绍开始使用 Git 前的相关知识。我们会先了解一些版本控制工具的历史背景，然后试着在你的系统上把 Git 跑起来，直到最后配置好，可以开始正常的开发工作。读完本章，你就会理解为什么 Git 会如此流行，为什么你真的需要使用它。
